Copy Hero

by Jeff Atwood

Copy Hero is a simple utility to convert a list of items, one on each line, to a list in a structured data format and copy it to your clipboard. This makes it easy to copy a list from a notes application or a spreadsheet and paste it into a Javascript script as a JSON array or in a SQL statement as a list of values.

The extension can be run either by manually pasting in the input list into the form input, or by using a shortcut to automatically read the list from your clipboard. The latter requires granting permission to Copy Hero to read from your clipboard the first time you use the shortcut. The default shortcuts are Alt+Shift+J to copy as JSON, and Alt+Shift+S to copy as SQL. On MacOS Alt is Opt.
