Newlines Converter

This tool converts lines of text to JSON or a SQL VALUES list and copies them to the clipboard.

Use-case scenarios

  • A customer success manager sends you a long list of IDs and wants you to query some information about them from the database. You need to paste them into SQL in the format (ID1, ID2, ID3, etc.).
  • You are debugging an issue in your SAAS application and you want to load some IDs from a spreadsheet into the browser console as an array so you can compare them with network responses. Therefore you need to convert the list into JSON so it can be pasted into the console without writing out parsing logic.

Get started

Paste your list into the input below or to have the extension automatically read the list from your clipboard.
